# NULLL_CLUB_CONTEXT.md

## Présentation

NULLL.CLUB est un Social Run Club basé à Aix-en-Provence.

Le sport est le prétexte. Le véritable objectif est de créer du lien, rapprocher les gens et construire une communauté forte autour d’expériences réelles.

Signature et idée centrale :

> MAKE IT REAL, RENDEZ LE REEL.

## Positionnement

NULLL.CLUB se situe à l’intersection de :

- la course à pied ;
- la communauté ;
- la culture ;
- la mode ;
- les rencontres ;
- la créativité ;
- les événements locaux.

Ce n’est pas un club de course traditionnel.

Le projet doit éviter les codes classiques du sport, du fitness, du bien-être institutionnel et des marques running trop lisses.

NULLL.CLUB doit sembler vivant, indépendant, culturel, humain et légèrement provocant/décalé.

## Mission

Créer des occasions concrètes de rencontrer de nouvelles personnes grâce au running.

Le sport sert de point de départ pour :

- sortir de l’isolement ;
- créer des rencontres spontanées ;
- développer une communauté locale ;
- partager des moments réels ;
- produire des événements et collaborations ;
- faire exister une culture propre à NULLL.CLUB.

## Public

Le public principal est composé de personnes situées à Aix-en-Provence et dans ses environs, notamment :

- jeunes adultes ;
- étudiants ;
- actifs ;
- créatifs ;
- coureurs débutants ou réguliers ;
- personnes cherchant à rencontrer du monde ;
- personnes attirées par la mode, la culture et les communautés alternatives.

Il n’est pas nécessaire d’être un athlète confirmé.

L’expérience doit rester accessible, inclusive et sociale.

## Ton éditorial

Le ton doit être :

- direct ;
- court ;
- humain ;
- confiant ;
- décalé ;
- contemporain ;
- légèrement provocant ;
- jamais institutionnel ;
- jamais trop publicitaire ;
- jamais artificiellement motivant.

Éviter :

- les phrases corporate ;
- les promesses exagérées ;
- le vocabulaire de coaching ;
- les clichés du développement personnel ;
- les textes génériques de marques sportives ;
- les expressions comme « dépasse tes limites » ou « deviens la meilleure version de toi-même ».

Privilégier :

- des phrases courtes ;
- des formulations mémorables ;
- des messages qui donnent envie de rejoindre un mouvement ;
- un vocabulaire simple ;
- une identité locale ;
- un sentiment d’appartenance.

## Direction artistique

La direction artistique est brutaliste et avant-gardiste.

Principes visuels :

- noir et blanc comme couleurs principales avec rose en tonique ;
- contraste fort ;
- typographies affirmées ;
- mise en page volontairement brute ;
- compositions décalées ;
- esthétique éditoriale ;
- références mode, culture et underground ;
- rythme visuel énergique ;
- identité reconnaissable.

Le design ne doit pas devenir :

- corporate ;
- minimaliste générique ;
- une landing page SaaS ;
- une interface fitness classique ;
- trop propre ou trop lisse ;
- coloré sans justification ;
- rempli de composants standards.

Les irrégularités peuvent être volontaires.

Elles ne doivent être corrigées que lorsqu’elles créent un vrai problème de lisibilité, de navigation ou de responsive.

## Site internet

Le site doit principalement permettre de :

1. comprendre rapidement ce qu’est NULLL.CLUB ;
2. découvrir les prochains runs ;
3. rejoindre la communauté ;
4. suivre NULLL.CLUB sur les réseaux ;
5. découvrir ou acheter le merchandising ;
6. renforcer le référencement local ;
7. donner une image crédible et distinctive du projet.

## SEO local

Le site doit pouvoir se positionner sur des recherches liées à :

- run club Aix-en-Provence ;
- running club Aix-en-Provence ;
- club de course Aix-en-Provence ;
- courir à Aix-en-Provence ;
- groupe running Aix ;
- run collectif Aix ;
- social run club Aix ;
- événements running Aix-en-Provence ;
- communauté running Aix ;
- course à pied Aix-en-Provence.

Le SEO doit rester naturel et ne pas dégrader les textes ou la direction artistique.

## Contenu

Ne jamais inventer :

- de dates de runs ;
- de lieux de rendez-vous ;
- d’horaires ;
- de prix ;
- de partenaires ;
- de sponsors ;
- de témoignages ;
- de chiffres de participation ;
- d’informations légales ;
- de stock produit.

Lorsqu’une donnée manque, conserver un placeholder clairement identifiable ou demander l’information uniquement si elle est indispensable.

## Merchandising

NULLL.CLUB peut proposer du merchandising, notamment des t-shirts.

Le merchandising doit prolonger l’identité du club :

- coupes modernes et boxy ;
- inspiration streetwear ;
- graphismes forts ;
- impression sérigraphique ;
- identité portable en dehors des runs ;
- cohérence avec la direction artistique du site.

Le merch ne doit pas ressembler à un simple vêtement technique de club sportif.

## Références générales

L’esprit peut se rapprocher de communautés running contemporaines comme Rawdawg, sans copier leur identité, leurs textes, leurs visuels ou leur structure.

NULLL.CLUB doit construire son propre langage et sa propre culture.

## Principe final

Chaque décision doit répondre à cette question :

> Est-ce que cela renforce une communauté réelle sans banaliser l’identité de NULLL.CLUB ?

Lorsque plusieurs solutions sont possibles, choisir celle qui :

1. préserve le mieux la direction artistique ;
2. améliore réellement l’expérience ;
3. reste simple techniquement ;
4. demande le moins de modifications ;
5. maintient de bonnes performances ;
6. réduit la dette technique.
